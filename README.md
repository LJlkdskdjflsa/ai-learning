# ai-learning

A collection of machine learning implementations and experiments.

## Contents

- **momentum_rmsprop_numpy_demo.ipynb**: Demonstrates gradient descent optimization algorithms (Vanilla GD, Momentum, RMSprop) using pure NumPy

## Setup

This project uses `uv` for virtual environment management.

### Install Dependencies

```bash
# Create virtual environment
uv venv

# Install dependencies
uv pip install numpy matplotlib jupyter
```

### Running Jupyter Notebooks

```bash
# Activate virtual environment
source .venv/bin/activate

# Start Jupyter Notebook
jupyter notebook
```
